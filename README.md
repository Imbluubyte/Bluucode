**Usage Instructions:**
Run ide.py and it's easy from there

**Syntax:**
printline(output) - Prints a line of output to the console.
add(x, y) - Adds the values of x and y and returns the answer.
sub(x, y) - Subtracts the values of x and y and returns the answer.
rng(x, y) - Returns a random integer between the values of x and y.
declare(var, val) - Declares a variable with the name var and assigns it a value of val.
